# Param's Poker Tracker 🃏

Minimalist web application to track poker game statistics and manage player performance.

## Features
- 👥 **Player Management**
  - Add and remove players
  - Track individual player statistics
  - View detailed player history

- 💰 **Money Tracking**
  - Record buy-ins and cashouts
  - Calculate net profits/losses
  - Track position finishes

- 🎲 **Group Management**
  - Create poker groups
  - Track group-specific games
  - Manage multiple game sessions

- 🏆 **Leaderboard**
  - All-time player rankings
  - Sorted by total earnings
  - Track win rates

Visit the live application: [Param's Poker Tracker](https://parameterization.github.io/params-poker-tracker/)
